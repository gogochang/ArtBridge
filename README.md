## 음악인들을 위한 커뮤니티 ArtBridge
* 음악인들의 소통을 위한 커뮤니티를 제공하는 서비스
* 악기를 배우고싶은사람, 악기를 가르치고 싶은사람들을 연결 시켜주는 서비스
* 악기 연주 영상 피드백

## 프로젝트 설명
* 기간 : 2024.04.20 ~
* 팀원 구성 : 김창규

## 개발환경
* 언어 : Swift
* UI : UIKit, SnapKit
* 라이브러리 : RxSwift, Kinghfisher
* Back-end: Supabase
* 아키텍처: MVVM-C
